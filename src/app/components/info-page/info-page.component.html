Lorenz Handschack Über das Projekt

Traffic Map Berlin ist eine Anwendung zur Visualisierung und Analyse von Verkehrs­behinderungen innerhalb Berlins. Dies umfasst Baustellen, Sperrungen, Unfälle und andere Störungen. Die entsprechenden Daten werden von der Verkehrs­informations­zentrale VMZ bereit­gestellt und können nach Zeitraum und Ursache gefiltert werden.

Dieses Projekt entstand im Sommer­semester 2020 im Rahmen des Moduls "C72 Projekt Software­entwicklung" an der HTW Berlin. Es wurde von einem studentischen Team bestehend aus einem Backend- und zwei Frontend-Entwicklern ohne nennenswerte Vor­erfahrungen im Bereich Web­developement erarbeitet.

Das Frontend wurde mit Angular 9.1.4 entwickelt und benutzt die Leaflet Bibliothek mit Markercluster Plugin für die Karte und Chart.js für die Statistiken. Das UI wurde mit Angular Material realisiert.

Das Backend wurde in Java entwickelt und nutzt Spring Data MongoDB um die Daten abzurufen, sowie Spring Data REST, womit die verarbeiteten Daten über eine REST-API bereitgestellt werden.
