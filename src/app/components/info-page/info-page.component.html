Lorenz Handschack Über das Projekt

Traffic Map Berlin ist eine Anwendung zur Visualisierung und Analyse von Verkehrsbehinderungen innerhalb Berlins. Dies umfasst Baustellen, Sperrungen, Unfälle und andere Störungen. Die entsprechenden Daten werden von der Verkehrsinformationszentrale VMZ bereitgestellt und können nach Zeitraum und Ursache gefiltert werden.

Dieses Projekt entstand im Sommersemester 2020 im Rahmen des Moduls "C72 Projekt Softwareentwicklung" an der HTW Berlin. Es wurde von einem studentischen Team bestehend aus einem Backend- und zwei Frontend-Entwicklern ohne nennenswerte Vorerfahrungen im Bereich Webdevelopement erarbeitet.

Das Frontend wurde mit Angular 9.1.4 entwickelt und benutzt die Leaflet Bibliothek mit Markercluster Plugin für die Karte und Chart.js für die Statistiken. Das UI wurde mit Angular Material realisiert.
